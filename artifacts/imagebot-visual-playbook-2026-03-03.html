🖼️ ImageBot Visual Playbook

Specialist lane for image-first tasks, with reusable generation workflow and fallback behavior.

Agent ID: imagebot Identity: ImageBot Model: gemini-3.1-pro-preview-customtools

Why this agent exists

  • Keep visual generation separate from general strategy/chat work.
  • Standardize prompt quality, model routing, and output validation.
  • Enable repeatable visual production without rebuilding workflow each time.

Current capabilities (v1)

  • Gemini image generation script copied into dedicated workspace.
  • Automatic key discovery from env / secrets / auth profile.
  • Reject invalid tiny outputs (<10KB) to avoid bad assets.
  • Can be orchestrated by Saber for image requests.

Core command

node /home/gagekappes/.openclaw/workspace-imagebot/scripts/generate_images.mjs \
  "gemini-3.1-flash-image-preview" \
  "cinematic neon skyline with soft fog and reflective streets" \
  "/mnt/chromeos/MyFiles/Downloads/SHARE/2026-03-03/imagebot-test.png"

How to call ImageBot from Saber

Use a dedicated sub-session when the task is visual-first:

Spawn imagebot → pass brief → return files + model/prompt metadata.

Recommended output contract per task:

  • 3 variants minimum
  • Prompt text and model IDs logged
  • Best pick + why
  • File paths ready for Remote Viewer artifacts
