Bluebook Citation App

A live GitHub Pages deployment of the Bluebook citation assistant Christopher started from Google AI Studio for Rose. The app currently centers on two flows: cite from pasted source text and cite from a web search, both powered client-side with a user-supplied Gemini API key.

Open Live App Open Repo
What’s there now
  • Bluebook Citation Assistant heading and legal-student focused framing.
  • Two working modes: Cite Source and Search & Cite.
  • API key manager in the UI so the user can store or clear their Gemini key locally in-browser.
  • Client-side Gemini integration rather than a server-backed architecture.
  • Static-host friendly deployment now configured through GitHub Pages Actions.
