What This Means Architecturally

This image lane is not a custom in-repo engine. The capability is installed inside the Linux/OpenClaw environment, while the generated assets are written into the shared project workspace. In other words:

  • Capability lives in Linux/OpenClaw
  • Outputs live in the shared workspace / repo
  • Invocation happens through the bundled skill script
