Primer Lane

The morning re-entry system, made explicit.

This page is the stable home for the Daily Session Primer system: the base markdown/spec that defines how primers should work, the live example for today, and the lens page that lets the same structure be biased toward different moods, needs, and kinds of mornings. This page now imports the shared Style One package.

How to use this lane

When you want a primer, the base instruction is: read the Daily Session Primer spec and generate today’s primer. If you want a particular tone or bias, add a lens — for example philosophical, strategic, motivational, gratitude, or restorative.

What this solves

It turns a loose morning ritual into a reusable collaboration protocol: one shared base structure, multiple useful lenses, and a cleaner way to request the style of re-entry you want.
