Styles

Visual languages we can reuse on purpose

This page now acts as a more neutral showroom. The background should stay mostly out of the way, while each style card carries its own visual identity. The goal is comparison: you should be able to glance down this page and immediately feel that these are genuinely different design systems.

Style 001 Dark / Cinematic / Glass

Luminous Glass Noir

The current Remote Viewer house style: dark atmospheric backgrounds, cyan highlights, muted amber warmth, translucent panels, rounded cards, and a premium editorial feel.
Style 002 Light / Print / Structured

Paper Editorial Modern

A deliberately different direction: bright paper-like surfaces, sharper hierarchy, cleaner contrast, restrained blue accents, and a more magazine/editorial or design-journal sensibility.
This is closer to the right pattern: the library page is a neutral display surface, while each card previews an actual visual language. If we keep building this, we can add more styles like brutalist monochrome, warm gallery minimalism, retro terminal, or playful experimental.
