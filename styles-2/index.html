Style 001 · House Style · Premium / Warm / Operational

Style One

The Remote Viewer house style: warm accent, calmer premium product-web composition, stronger hierarchy, and a clearer standard for hosted operational pages.

This card is styled as itself. It is both the specimen and the current house grammar for standard pages.

Hero posture Split + offset

Accent rule One warm accent

Page role System coherence

A calm document-viewer system for long markdown pages, imported notes, reports, and internal writing that should feel native in Remote Viewer without being forced into a more theatrical artifact treatment.
